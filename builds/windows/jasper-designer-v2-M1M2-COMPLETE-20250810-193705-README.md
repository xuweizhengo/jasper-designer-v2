## 🚀 使用方法

### 快速启动
1. 解压 `jasper-designer-v2-M1M2-COMPLETE-20250810-193705.tar.gz`
2. 双击运行 `run-jasper.bat` 或直接运行 `jasper-designer.exe`
3. 开始设计体验！

### 核心操作
- **创建元素**: 从左侧组件库拖拽到画布
- **选择元素**: 点击单选，空白区域拖拽框选多个
- **移动元素**: 拖拽选中元素到新位置  
- **Ctrl多选**: 按住Ctrl点击切换元素选择状态
- **清除选择**: 点击空白区域
